# NightVault v0.9.9 — установщик Windows

## Что делает установщик

После сборки получится файл:

```text
NightVault-Setup-0.9.9.exe
```

Он работает как обычный установщик Windows:

- выбор папки установки;
- ярлык на рабочем столе;
- ярлык в меню Пуск;
- запуск клиентской части NightVault;
- Electron и runtime уже внутри приложения, поэтому пользователю не нужно отдельно ставить Node.js/npm для клиента.

## Важно

Node.js/npm нужны только разработчику, чтобы **собрать** установщик из исходников. Пользователю, который запускает готовый `NightVault-Setup-0.9.9.exe`, Node.js/npm не нужны.

## Как собрать установщик

На Windows запусти:

```cmd
build-installer.bat
```

Или вручную:

```cmd
npm install
npm run build:installer
```

Готовый `.exe` будет в папке:

```text
dist
```

## Сервер

Ярлык запускает только клиентскую часть, как ты просил.

Сервер для теста можно запускать отдельно:

```cmd
npm run server
```

Для друзей через Radmin VPN в клиенте указывается адрес сервера вида:

```text
http://26.x.x.x:3000
```
