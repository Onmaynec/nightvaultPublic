# NightVault v0.9.7

## Главное

В этой версии исправлена и усилена система обновлений, чтобы она была понятной пользователю и ближе к Telegram Desktop.

## Новое

- В разделе **Опции** добавлен блок **О приложении**.
- Теперь в настройках видно установленную версию NightVault.
- Добавлена большая кнопка **Проверить обновления**.
- Кнопка проверки обновлений добавлена также на экран входа.
- При запуске установленного клиента проверка обновлений запускается автоматически.
- Если доступна новая версия, появляется красивая модалка с требованием обновления.
- После скачивания обновления появляется окно **Перезапустить и установить**.
- После установки новой версии показывается окно **Что нового** один раз.

## Исправлено

- Событие `update-available` теперь не должно теряться из-за слишком ранней проверки.
- Ошибки автообновления теперь показываются нормальным окном, а не только коротким toast.
- В dev-режиме приложение понятно пишет, что автообновления работают только в установленной сборке.

## Как выпускать эту версию

```cmd
git add .
git commit -m "NightVault v0.9.7"
git push origin main
git tag v0.9.7
git push origin v0.9.7
```

После успешной сборки GitHub Actions автоматически создаст релиз с `latest.yml` и `.exe` файлами.
